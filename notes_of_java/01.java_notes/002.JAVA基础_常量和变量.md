## 1. 常量 
- 定义: 在程序中固定不变化的值        
- 常量的分类:  
  - 字面值常量: 整数常量, 小数常量, 布尔常量等
  - 定义的 final 变量

## 2. 变量
- 定义: 
  表示存储空间, 可以用来存放某一类型的常量,  
  没有固定值, 可重复使用, 可以存放某种类型的未知数据

- 特点:
  1. 占据着内存中的某一块存储区域
  2. 该区域有自己的名称(变量名)和类型(数据类型)
  3. 可以被重复使用
  4. 该区域的数据可以在同一类型范围内不断变化

## 3. 变量的分类, 作用域, 使用规则

### 3.1. 变量的分类
- 随着变量定义在类中的位置不同, 把变量分成两大类:
  - 成员变量: 直接定义在类里的变量, 又称为全局变量或字段
  - 局部变量: 除了成员变量外, 其它都是局部变量.  
    根据定义的位置不同有 3 种表现形式:  
    方法形参, 方法内变量, 代码块变量 

### 3.2. 变量的作用域
- 指变量的存在范围, 只有在这个范围内, 程序代码才能访问它.  
  当一个变量被定义时, 它的作用域就确定了  
  作用域是定义开始到定义所在的花括号结束

### 3.3. 变量使用规则
- 变量必须先声明, 并且初始化后才能使用
- 声明变量必须有数据类型
- 同一作用域内变量不能重复定义
